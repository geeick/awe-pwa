# Grounded PWA

A React + Vite progressive web app.

## Run locally

```bash
npm install
npm run dev
```

Open the local URL shown by Vite.

## Test the installable PWA

```bash
npm run build
npm run preview
```

Open the preview URL in Chrome or Edge. Use the browser's Install App option. Service workers require localhost or HTTPS.

## Included interactions

- Mood selection persists in local storage
- Practice completion persists in local storage
- Functional bottom navigation state
- Install prompt when supported
- Offline app-shell caching
- Responsive phone-sized layout
